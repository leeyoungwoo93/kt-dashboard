# KT Dashboard Improvement Patch

## 핵심 반영 사항

1. 소매 매장실적 탭 신설
   - 12번째 탭: `⑫ 소매 매장실적`
   - 매장/대리점/담당·지사/본부/채널 기준 집계 전환
   - 월별 소매 추이, 채널 구성, TOP 차트, 성과 믹스, 상세 테이블 제공

2. 소매 업로드 기능 복구
   - 상단 업로드 버튼에 `🏪 소매` 추가
   - `/upload/storesales`와 프론트엔드 파일 input 연결
   - 기존 파일 업로드 input들도 JS 이벤트 핸들러로 연결

3. 월별 데이터 안정화
   - 판매 데이터 `yyyymm` 파싱을 `YYYYMM`으로 정규화
   - `202601`, `2026-01`, 엑셀 날짜/Timestamp 계열 모두 대응
   - 소매 매장실적도 기준월 필터와 연동

4. 탭 전환 오류 방지
   - 기존 DOM 순서 기반 탭 전환을 `tab0`, `tab1`... id 기준으로 변경
   - 신규 탭 추가 시 기존 탭이 밀려서 빈 화면이 되는 문제 방지

5. 소매 매장실적 원천 양식 대응
   - `본부 코드/본부명`, `판매접점 코드/판매점명`처럼 코드와 명칭이 나란히 있는 파일 구조 대응
   - 매장명이 비고 코드만 보이던 문제 방지

6. 자동 로드 경로 보강
   - `app/data/*.xlsx` 우선 로드
   - 기존처럼 `app/*.xlsx`에 들어간 파일도 자동 로드되도록 fallback 추가
